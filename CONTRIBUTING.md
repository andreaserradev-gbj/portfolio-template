# Contributing

Thanks for your interest in contributing to this portfolio template!

## Quick Start

```bash
# Fork and clone the repo
git clone https://github.com/YOUR_USERNAME/portfolio-template.git
cd portfolio-template

# Install dependencies
npm install

# Start dev server
npm run dev
```

## Making Changes

1. Create a branch: `git checkout -b my-feature`
2. Make your changes
3. Run tests: `npm run test:run`
4. Commit and push
5. Open a PR against `main`

## Code Style

Code is auto-formatted on commit via pre-commit hooks (ESLint + Prettier). Just write code and commit - formatting happens automatically.
